문제

사용자가 가위(1), 바위(2), 보(3) 중 하나를 숫자로 선택하지 않으면 다시 안내하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 4
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
2를 냈습니다. 


가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
1를 냈습니다. 


가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 10
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 8
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 6
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 4
가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
3를 냈습니다.

코드 빈칸 채우기

#include <stdio.h> 
int main() { 
	int player = 0;		 
	 {	  
		printf( "가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 
		scanf( "%d", &player ); 
	 
	}  ( ( player  1 )  ( player  2 )  ( player  3 ) );	  
	 
	printf( "%d를 냈습니다.\n", player );	 
	 
	return 0; 
}
