문제

크기를 입력받고 그 크기에 맞는 삼각형을 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
삼각형 크기를 입력하세요 : 3
  *
 ***
*****


삼각형 크기를 입력하세요 : 5
    *
   ***
  *****
 *******
*********


삼각형 크기를 입력하세요 :0

코드 빈칸 채우기

#include <stdio.h> 
 
int main() { 
	int size = 0, line = 0, blank = 0, star = 0; 
 
	printf("삼각형 크기를 입력하세요 : "); 
	scanf("%d", &size ); 
 
	 ( line = 1; 
			line  size;						 
				      line    ) { 
		 ( blank = size - line;					 
				      blank  0; 
					         blank    ) { 
			printf(" "); 
		} 
		 ( star = 2*line-1;					 
				      star  0; 
						star    ) { 
			printf("*"); 
		} 
		printf("\n"); 
	} 
 
	return 0; 
}
