문제

시작요일과 말일을 입력받아 달력을 출력하는 문제입니다 실행순서를 클릭하세요 
일(0),월(1),화(2),수(3),목(4),금(5),토(6) 중에서 
이번달 1일은 무슨요일인가요 : 2
이번달 말일은 언제인가요 : 9
   일   월   화   수   목   금   토
              1    2    3    4    5
    6    7    8    9

프로그램 코드

	#include <stdio.h>


1	int main() {

		int firstDay = 0;

		int lastDay = 0;

		int day = 0;

		int dayOfWeek = 0;

		int sunday = 0;

2		int saturday = 6;


3		printf( "일(0),월(1),화(2),수(3),목(4),금(5),토(6) 중에서 \n이번달 1일은 무슨요일인가요 : " );

4		scanf("%d", &firstDay );

5		printf( "이번달 말일은 언제인가요 : " );

6		scanf("%d", &lastDay );


7		printf( "   일   월   화   수   목   금   토\n");


8		for( dayOfWeek = sunday;

9		                        dayOfWeek < firstDay;

10		                                             dayOfWeek++ ) {

11			printf( "     " );

		}

		

12		for ( day = 1;

13		              day <= lastDay;

14		                             day++, dayOfWeek++ ) {

15			printf( "%5d", day );

16			if ( ( dayOfWeek % 7 ) == saturday ) {

17				printf( "\n");

			}

		}


18		return 0;

	}
