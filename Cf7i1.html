문제

정수를 입력받아 0부터 입력값까지의 합계 Σ를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
숫자를 입력하세요: 3
Σ 3 = 3 + 2 + 1 + 0 = 6


숫자를 입력하세요: 5
Σ 5 = 5 + 4 + 3 + 2 + 1 + 0 = 15


숫자를 입력하세요: 0
Σ 0 = 0 = 0

알고리즘

프로그램 시작 
   숫자 n 입력받기 
   n부터 0까지의 합계 계산하기 
   합계 출력하기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter07/Summation.c

	#include <stdio.h>

	 

	// 프로그램 시작 

1	int main() {

2		int num = 0, sum = 0;

	 

		// 숫자 n 입력받기 

3		printf( "숫자를 입력하세요: ");

4		scanf( "%d", &num );

		 

		// n부터 0까지의 합계 계산하기 

5		printf( "Σ %d = ", num );

		for( ;

6			num > 0;

7				num-- ) {

8			printf ( "%d + ", num );

9			sum += num;

		}

		 

		// 합계 출력하기 

10		printf ( "%d = %d", num, sum );

	 

		//프로그램 종료 

11		return 0;

	}
